## 1. 프로젝트 소개

### 부동산 안심 계약 플랫폼 "전 새내기"
#### 전 : 전 문가처럼 등기부등본을 분석하고, 
#### 새 : 새로운 기준으로 계약서를 꼼꼼히 비교해요. 
#### 내 : 내 불안했던 부동산 거래는 이제 안심!
#### 기 : 기존의 걱정을 싹 지워주는 앱, 바로 우리 전 새내기랍니다!
